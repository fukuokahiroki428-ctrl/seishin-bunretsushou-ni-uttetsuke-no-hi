사용 안내
1. 시작 누르면 별도 Chrome 창이 뜸 (제어용 CDP 연결).
2. 로그인이나 캡챠는 Chrome 창에서 직접 처리. 한번 로그인하면 다음 실행 때도 유지(영구 프로필).
3. 페이지 도달했으면 좌측 캡쳐 버튼 — SingleFile로 인라인 HTML 저장 (이미지/스타일 포함).
4. 끝까지 스크롤은 lazy-load 강제 트리거용.
5. 미러 자동 시도는 죽은 사이트 후보군에서 살아있는 미러 자동 탐색.
