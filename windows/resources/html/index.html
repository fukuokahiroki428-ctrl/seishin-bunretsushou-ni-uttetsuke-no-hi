디스크 설정
보조 저장 경로 (대용량 다운 자동 분산)

각 플랫폼의 1번 저장 경로(예: 외장 SSD)가 가득 차면 자동으로 이 경로로 전환합니다. 남은 공간 10GB 미만이면 보조 경로 사용. 비워두면 비활성.

계정 토큰 자동 추출

로그인된 쿠키/토큰을 자동 추출합니다. Bluesky는 앱 비밀번호 필요 → 수동 입력만 가능.

🍎 macOS 사용자: Chrome / Firefox / LibreWolf / Edge / Brave / Arc / Vivaldi 모두 시도합니다.
★ Chrome 키체인 prompt 뜨면 "항상 허용" 누르세요 — 다음부터는 자동.
★ 거부했으면 다시 ⌘ 스페이스 → 키체인 접근 → login → Chrome Safe Storage 항목 권한 변경.

모듈 / 라이브러리 업데이트

번들 Python 패키지를 최신 버전으로 업데이트합니다 (twikit, httpx, atproto 등).

Python 환경 관리

🛠 yt-dlp 자동 업데이트 (⚠ 보안 위험)

⚠ 켜기 전 읽어주세요: yt-dlp 가 GitHub 에서 새 버전을 받습니다. GitHub 저장소가 변조될 경우 (드물지만) 악성 binary 가 다운로드될 위험. yt-dlp 자체 SHA256 검증과 우리 sanity check (--version 정상 출력) 가 있지만 100% 안전 보장은 아님.
꺼져 있으면 앱과 함께 출하된 검증된 yt-dlp 만 사용 (안전, 단 Twitter/Instagram 포맷 변경 시 깨질 수 있음 → 그때 새 DMG 받기).

📦 NAS 자동 백업 (로컬 → NAS 폴더 복사)

다운로드/캡쳐 완료된 파일을 백그라운드로 NAS 마운트 폴더에 자동 cp.

🚀 속도 자동 최적화:
• Apple WebDAV mount + WebDAV 자격증명 → rclone 자동 사용 (5-50배 빠름)
• Mountain Duck mount → 8 워커 + cache 활용 (background sync)
• SMB / AFP / NFS / 외장 SSD → 4 워커 병렬 cp
★ NAS 가 본인 소유 아니면 Mountain Duck (mountainduck.io) 권장

📦 저장 위치 모드

둘 다 꺼두면 각 플랫폼 탭에서 개별 경로 지정. 체크하면 모든 플랫폼 input 자동 숨김.

Finder에 마운트: macOS가 OS 차원에서 인증 처리 → 권한 거부 우회. 마운트되면 /Volumes/<공유폴더> 가 생기고 Finder 사이드바에 표시. 저장 경로를 그 폴더로 바꾸면 다운로드가 NAS로 직행.
